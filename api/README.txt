O Na hora certa é uma ferramenta de aluguel de sala comercial 
desenvolvida em Python usando o framework Flask.

-----------------------------------------------

Instalação

* Certifique-se de ter o Python 3.8+ instalado em sua máquina.

1) Navegue até o diretório do projeto: cd {caminho até a pasta da aplicação}/na-hora-certa
2) Instala as dependências com o comando: pip install -r ./api/requirements.TEXT

Execução:

* Certifique-se de ter o SQLite instalado em sua máquina. 

1) Execute a aplicação por meio do comando: python api/app.py
2) Acesse a URL http://localhost:5000/swagger ou http://127.0.0.1:5000/swagger.

No Swagger você encontrará informações sobre os endpoints e os modelos de informações
necessários para realizar as chamadas.
